📲 Come installare le app

Le nostre app si scaricano come file APK — non dal Play Store. Dopo il download troverai il file nella cartella Download del tuo telefono. Ecco come procedere a seconda del tuo dispositivo.

📱 Samsung (One UI)

  • 1 Apri l'app Archivio (o Gestione file)
  • 2 Vai nella cartella Download
  • 3 Tocca il file TraduttoreVocaleV1.apk
  • 4 Se richiesto, tocca Impostazioni e abilita Installa app sconosciute
  • 5 Torna indietro e tocca Installa

📱 Xiaomi / Redmi (MIUI)

  • 1 Apri il Gestore file
  • 2 Vai nella cartella Download
  • 3 Tocca il file TraduttoreVocaleV1.apk
  • 4 Tocca Installa — potrebbe chiedere di autorizzare il browser o il gestore file
  • 5 Conferma e attendi l'installazione

📱 Huawei / Honor

  • 1 Apri File dal menu principale
  • 2 Vai in Download
  • 3 Tocca il file .apk
  • 4 Se richiesto, vai nelle impostazioni e abilita Origini sconosciute
  • 5 Torna e tocca Installa

📱 Android standard (Google Pixel, Motorola…)

  • 1 Apri l'app File di Google
  • 2 Tocca Download in basso
  • 3 Tocca il file TraduttoreVocaleV1.apk
  • 4 Tocca Continua e poi Installa
  • 5 L'app appare subito tra le tue applicazioni
💡 Il messaggio "App da fonte sconosciuta" è normale: significa solo che l'app non viene dal Play Store. Le nostre app sono sicure, indipendenti e senza tracciamenti. Puoi installare in tutta tranquillità.
